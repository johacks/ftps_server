Create a socket for a server given the protocol to use, port and the queue buffer size. Performs the bind (and the listen if applicable).

Based on: https://www.tenouk.com/Module40a.html protoent info: https://linux.die.net/man/3/getprotobyname info about sockaddr: https://www.gta.ufrj.br/ensino/eel878/sockets/sockaddr_inman.html info about socket(): man socket info about bind(): man 2 bind info about listen(): man 2 listen

Parameters
proto_transpString representing the name of the protocol: eg "udp", "tcp"... If NULL or incorrect, use TCP
qlenSocket queue buffer size
portPort number where the socket will be opened
ip_srvIP of the server
Returns
int (file descriptor of the created socket)
